## 기능 구현 목록

- [x] 요금을 1000원 이상 투입해야 한다.
  - ex) 1000 -> success
  - 500 -> error
- [x] 요금을 1000으로 나눈 개수만큼 로또를 구매한다.
  - ex) 5400 -> 5
- [x] 중복되지 않는 1 ~ 45사이의 숫자를 6개 생성한다.
  - ex) [1, 2, 3, 4, 5, 6] -> success
  - ex) [100, -1] -> error
- [x] 생성된 로또를 렌더링한다.
- [x] 번호 보기 토글 버튼을 클릭해 로또 번호를 볼 수 있어야 한다.
